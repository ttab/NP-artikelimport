Innehållet i denna ZIP är exempel på kod och xslt för att konvertera TT-artiklar till NPexchange dokument som kan importeras som artiklar i Newspilot.

Det är fritt fram att använda dessa xslt-filter och kod-exempel för eget bruk. De kan dock behöva kompletteras för att matcha värden för avdelningar, id-koder mm som används i den avsedda Newspilot-installationen.
